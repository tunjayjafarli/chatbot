# chatbot
Automated Telegram bot powered by AI

### Set up virtual environment
MAC:
```
$ python3 -m venv venv
$ source venv/bin/activate
```

WINDOWS:
```
PS> python -m venv venv
PS> venv\Scripts\activate
```

### Install Dependencies
```
$ pip install -r requirements.txt
```

### Run the bot script
```
$ python boy.py
```
